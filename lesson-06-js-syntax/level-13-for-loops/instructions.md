# Level 13 — `for` loops practice (10 different ranges)

## Objective

Write 10 different `for` loops. Each loop must use a different start/end range and display each iterated number with `console.log`.

## Benefits

- Reinforces loop structure and off-by-one thinking.

## Complete these tasks

- Implement 10 `for` loops.
- Each loop should have different start/end ranges.
- Use `console.log` in each `for` to display each count.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Pattern: `for (let i = 1; i <= 5; i++) { console.log(i); }`.

## More information

- Vary start and end values to practice inclusive/exclusive conditions.

## Usage tips

- Test each loop separately to confirm output.

## Example

```js
for (let i = 0; i < 3; i++) {
  console.log(i);
}
```
