# Level 01 — Declare 10 variables (use `let`)

## Objective

Declare ten separate variables using with NO initial value (no `=`). Do not use `var`.

## Benefits

- Builds familiarity with variable declaration syntax.
- Prepares for later initialization and assignment tasks.

## Complete these tasks

- Add 10 `let` declarations, one per line, with descriptive camelCase names.
- Leave each variable uninitialized.
- Export one of the declared variables as the `default` export so unit tests can inspect a submitted value.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Remember `let name;` declares without initializing.
- Avoid `const` (it requires an initial value).

## More information

- `let` creates block-scoped variables. A declaration with no `=` leaves the value `undefined`.

## Usage tips

- Type each line manually to build muscle memory.

## Example

```js
// correct form (one of ten lines):
let userName;
```
