# Level 12 — Debug function declarations and calls

## Objective

Fix five broken function declarations and their corresponding calls so each declaration and call parse and run logically.

## Benefits

- Builds skills finding common function-related syntax errors.

## Complete these tasks

- Repair five broken function declarations and their example calls.
- After repairing functions and calls, export one repaired function as the `default` export so unit tests can import and inspect it.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Check for missing parentheses, commas between params, and mismatched braces.

## More information

- Function syntax: `function name(params) { ... }` or `const name = (params) => { ... }`.

## Usage tips

- Fix one function and its call, run, then move to the next.

## Example

```js
function greet() {
  return "hello";
}
greet();
```
