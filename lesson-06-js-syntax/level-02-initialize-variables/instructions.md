# Level 02 — Initialize descriptive string and number variables

## Objective

Declare and initialize 10 variables on the same line: 5 descriptive strings and 5 descriptive numbers.

## Benefits

- Practice literal syntax for strings and numbers.
- Reinforces descriptive camelCase naming.

## Complete these tasks

- Add 5 string variables and 5 number variables, declared and initialized on the same line.
- Use camelCase names like `userName` and `totalCount`.
- Export one of the declared variables as the `default` export so unit tests can inspect a submitted value.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Strings need quotes: `let name = "Ava";`.
- Numbers are plain: `let total = 42;`.

## More information

- Use descriptive names to make code self-documenting.

## Usage tips

- Keep one declaration per line; don’t compress multiple variables on one line for this exercise.

## Example

```js
let userName = "Ava";
let totalCount = 10;
```
