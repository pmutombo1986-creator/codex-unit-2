# Level 07 — Declare 10 objects with properties

## Objective

Declare and initialize 10 object variables using descriptive camelCase names. Each object must have at least 3 properties (mix of strings, numbers, booleans).

## Benefits

- Practice object literal syntax and property notation.

## Complete these tasks

- Create 10 single-line object literals with descriptive names and >=3 properties each.
- Export one of the created objects as the `default` export so unit tests can inspect a submitted value.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Use `:` between property names and values and `,` between properties.

## More information

- Object literals use `{ key: value }` syntax.

## Usage tips

- Keep objects short and varied for practice.

## Example

```js
let userProfile = { name: "Ava", age: 28, isSubscriber: false };
```
