# Level 03 — Assign values and debug string initializations

## Objective

Part A: Assign values to five previously declared variables (each on its own line). Part B: Fix five faulty string initializations provided in the starter file.

## Benefits

- Practice assignment separate from declaration.
- Learn to identify common string mistakes.

## Complete these tasks

- Assign values to five uninitialized variables on separate lines.
- Repair five broken string initializations so they parse.
- Export one of the assigned or repaired variables as the `default` export so unit tests can inspect a submitted value.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Check for missing quotes, mismatched quotes, or missing `=`.
- Fix one error at a time and run the file to confirm.

## More information

- Strings require matching single or double quotes; escaping is not required for these simple examples.

## Usage tips

- Use `console.log` to verify assigned values while debugging.

## Example

```js
let firstName;
firstName = "Ava"; // assignment on a separate line
```
