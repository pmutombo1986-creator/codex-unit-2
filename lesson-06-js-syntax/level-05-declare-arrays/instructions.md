# Level 05 — Declare string and number arrays (same-line, plural names)

## Objective

Declare and initialize 10 arrays on the same line: 5 plural-named string arrays and 5 plural-named number arrays. Each array must have at least 3 items.

## Benefits

- Practice array literal syntax and naming conventions.

## Complete these tasks

- Add 5 string arrays and 5 number arrays with plural camelCase names (e.g., `userNames`).
- Each array should contain at least 3 items.
- Export one of the arrays as the `default` export so unit tests can inspect a submitted value.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Arrays use square brackets: `let names = ["Ava", "Jon", "Maya"];`.

## More information

- Keep array names plural to indicate a collection.

## Usage tips

- Type the brackets, commas, and quotes carefully; missing commas are common errors.

## Example

```js
let scores = [10, 20, 30];
```
