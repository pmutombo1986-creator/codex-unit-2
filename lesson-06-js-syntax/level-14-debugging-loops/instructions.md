# Level 14 — Debugging broken loops

## Objective

Repair provided loop snippets so they parse and run, printing the intended sequences with `console.log`.

## Benefits

- Improves ability to find loop-related syntax mistakes.

## Complete these tasks

- Fix 10 broken loop snippets to valid `for` loops that output the intended ranges.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Check for missing parentheses, missing increments, and incorrect braces.

## More information

- Loop correctness includes start value, condition, and increment/decrement.

## Usage tips

- Run each fixed loop to verify its output before moving on.

## Example

```js
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```
