# Level 06 — Debug array declarations

## Objective

Fix common array syntax errors so each array is declared and initialized on one line and has at least 3 items.

## Benefits

- Practice identifying missing brackets, commas, and other array mistakes.

## Complete these tasks

- Repair 10 provided array snippets to valid one-line array declarations with >=3 items.
- After repairing arrays, export one of the fixed arrays as the `default` export so unit tests can inspect a submitted value.
- Write each line from scratch; do NOT copy & paste any code. Typing each line reinforces syntax memory and debugging skills.

## Hints

- Look for missing commas between items and unmatched brackets.
- Ensure strings have matching quotes.

## More information

- Arrays are created with `[]`; items are comma-separated.

## Usage tips

- Fix one snippet at a time and run the file to verify it parses.

## Example

```js
let fruits = ["apple", "banana", "cherry"];
```
