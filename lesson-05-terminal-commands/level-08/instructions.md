## Level 08 — Clear the terminal

### Objective

Clear the terminal screen using `clear` or `Ctrl+L` and explain when that's useful.

### Benefits

- Keeps your terminal readable when output gets long.

### Complete these tasks

- Run `clear` or press `Ctrl+L` to clear the terminal.
- Save a short string or array in `answer.js` that mentions `clear` or `ctrl+l`.

### Hints

- `Ctrl+L` is a quick keyboard shortcut that works in most terminals.

### More information

- Clearing doesn't delete history — you can still scroll up in many terminals.

### Usage tips

- Clear before running a long script so you can see only the new output.
