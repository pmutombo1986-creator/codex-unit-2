## Level 05 — Parent directory `..`

### Objective

Use `..` to move up to a parent folder and reference sibling files.

### Benefits

- Helps navigate projects with nested folders.

### Complete these tasks

- From a subfolder run `cd ..` to move up, then reference a sibling file with `../`.
- Save a short note in `answer.js` mentioning `..` or "parent directory".

### Hints

- `../sibling/file` accesses a file in a sibling folder.

### More information

- `..` is a directory name recognized by the shell meaning the parent folder.

### Usage tips

- Combine `pwd` and `ls` to confirm where you are.

// export default "used .. to move to parent"
