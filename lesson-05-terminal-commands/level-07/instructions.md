## Level 07 — Command history (Up/Down arrows)

### Objective

Use the Up/Down arrow keys and `Ctrl+R` to find past commands and rerun them.

### Benefits

- Saves time by reusing previous commands.

### Complete these tasks

- Press the Up arrow to find a recent command, then press Enter to run it.
- Optionally try `Ctrl+R` to search your command history.
- Record a short note in `answer.js` mentioning "up"/"down" or "ctrl+r".

### Hints

- `Ctrl+R` searches as you type — keep pressing it to find older matches.

### More information

- History is stored per-shell session; restarting may clear it.

### Usage tips

- Use history to avoid retyping long commands.
