## Level 06 — Run Node.js files with `node`

### Objective

Run a JavaScript file with the `node` command and observe the output.

### Benefits

- Lets you run scripts and see console output immediately.

### Complete these tasks

- Create or open a small `hello.js` file and run `node hello.js`.
- Save a short note in `answer.js` that mentions `node`.

### Hints

- Check for syntax errors if the script doesn't run.

### More information

- `node file.js` runs the file using Node.js installed on your computer.

### Usage tips

- If `node` isn't found, ensure Node.js is installed and on your PATH.

### Example

```js
// export default ["node", "hello.js"]
```
