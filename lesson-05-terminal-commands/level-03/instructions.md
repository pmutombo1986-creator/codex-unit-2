## Level 03 — Change directory with cd

### Objective

Use `cd` to move into subfolders and back to parent folders.

### Benefits

- Lets you work inside project folders without leaving the terminal.

### Complete these tasks

- Use `ls` to find a folder, `cd` into it, then use `cd ..` to go back.
- Save a short string in `answer.js` describing what you did.

### Hints

- `cd ..` always moves one level up.

### More information

- `cd folderName` moves you into `folderName` if it exists.

### Usage tips

- Tab-completion helps type folder names quickly.

### Example

```js
// export default "cd into level-03-folder and back"
```
