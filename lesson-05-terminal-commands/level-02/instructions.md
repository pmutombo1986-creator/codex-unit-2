## Level 02 — List files with ls

### Objective

Use `ls` to list files and folders in the current directory.

### Benefits

- Quickly see what's in a folder.
- Helps find files you need to open or run.

### Complete these tasks

- Open the integrated terminal and run `ls`.
- Note at least two items you see and save a short description in `answer.js`.

### Hints

- Folders appear in a diferent color and end with `/`.

### More information

- `ls` shows file and folder names in the current directory.

### Usage tips

- Use `pwd` to confirm your current folder.

```js
// export default "I ran ls and saw: file1, folderA"
```
